StickyScroll Fluid Demo

So your sidebar can follow anywhere you go (now with fluid support)!

Explanation: In this demo, the two columns are wrapped in a div.container, and we want the sidebar to stay within, so we the code used is $('.sidebar').stickyScroll({ container: '.container' }).

Important: the container element must be clearfixed, otherwise its height is reported as 0 and all of this plugin's calculations fail.

What about elements that don't have such a convenient container? See the demo of 'manual' mode.
