StickyScroll Demo

So your sidebar can follow anywhere you go

Explanation: Notice that the sidebar and main section do not share an exclusive container. For this situation, StickyScroll needs to be called the options $('.sidebar').stickyScroll({ mode: 'manual', bottomBoundary: 100 }) where bottomBoundary is the pixel distance away from the bottom of the document at which you wish the sticky element to stop. Whew, that was a mouthful. To make it a bit simpler, in this example, we set it to 100 pixels because that is the height of the footer, and we don't want it spilling over into the footer, right?

No topBoundary is required, the plugin detects its original position automatically.

If your sticky element has a parent element you need it to be contained to, you can use the automatic mode instead. It's much more convenient. See the demo page.
